## ゲームルール
操作はコントローラのみ(現在DUALSHOCK4でのみ操作可能)
お互いに4種のキャラクターと、4種の特殊攻撃を1つずつ選び
先に相手のHPを0にしたプレイヤーの勝利


## 操作方法
###キャラクター選択画面
L1･R1ボタンでキャラクター、特殊攻撃の種類を変更
〇ボタンで決定

###対戦画面
Lスティック/移動
Rスティック右/特殊攻撃の射出方向時計回り
Rスティック左/特殊攻撃の射出方向反時計回り
〇ボタン/通常攻撃
R1/特殊攻撃
L1/ガード
